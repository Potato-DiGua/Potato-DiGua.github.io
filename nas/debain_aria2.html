debian 12安装aria2并配置webui ​

| 环境基于debian 12

新建aria2用户 ​

shell
sudo adduser --system --no-create-home --group aria2user

创建aria2配置 ​

在/etc/aria2/aria2.conf中写入以下内容

dir=/mnt/mydisk/downloads
max-concurrent-downloads=4
continue=true
disable-ipv6=true
enable-rpc=true
rpc-allow-origin-all=true
rpc-listen-all=true
rpc-listen-port=6800

创建aria2服务 ​

在/etc/systemd/system/aria2.service中写入以下内容

[Unit]
Description=Aria2c Download Manager
After=network.target

[Service]
ExecStart=/usr/bin/aria2c --conf-path=/etc/aria2/aria2.conf
Restart=on-failure
User=aria2user
Group=aria2user

[Install]
WantedBy=multi-user.target

启动服务

shell
# 启动服务
sudo stystemctl start aria2
# 开机自启动
sudo stystemctl enable aria2

创建下载文件夹 ​

shell
mkdir /mnt/mydisk/downloads
sudo chown -R aria2user:aria2user /mnt/mydisk/downloads

安装web界面 ​

shell
# 安装nginx
# 下载AriaNg
wget https://github.com/mayswind/AriaNg/releases/download/1.3.10/AriaNg-1.3.10-AllInOne.zip
# 解压
unzip ./AriaNg-1.3.10-AllInOne.zip -d aria2
# 移动到nginx的目录下
sudo mkdir /var/www/html/aria2
sudo cp ./aria2/index.html /var/www/html/aria2/
