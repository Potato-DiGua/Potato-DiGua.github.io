Android与Web尺寸单位 ​

  • Anroid使用dp（基于像素密度的抽象单位，1dp代表的像素长度=density(像素密度) * 1 px，在每英寸160像素点的手机上1dp=1px）适配。同样尺寸上不同分辨率（不同density）的设备，每1dp所代表的像素数量是不一样的。但是所表示的物理长度（160dp=1in）是一样的。
  • h5使用rem(是指相对于根元素的字体大小的单位)适配，一般1 rem=screenWidth/750*100，利用屏幕宽的比值实现屏幕适配，7.5rem一定能充满屏幕的宽度，不是代表物理长度。
  • css中的px具有与Android中的dp等同的效果
